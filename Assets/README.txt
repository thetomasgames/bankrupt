Projeto simples escrito em C# para ser executado por linha de comando.

Para compilar e executar:
Mac/Linux:
Requer a instalação do Mono (http://www.mono-project.com/)

compiar: mcs /out:Bankrupt.exe *.cs
executar: mono Bankrupt.exe

Windows:
(abrir o prompt de de comando para Visual Studio)
compilar: csc /out:Bankrupt.exe *.cs
executar: Bankrupt.exe
